# Student Manager (JPA / Hibernate)

Console-based application for managing students using JPA and Hibernate.

## Features

- Add student
- List students
- Find by ID
- Update student data
- Delete student
- Database persistence (H2)

## Technologies

- Java
- JPA (Hibernate)
- H2 Database
- Maven

## Concepts practiced

- @Entity mapping
- EntityManager
- Transactions
- CRUD with database
- DAO pattern
- Persistence configuration
